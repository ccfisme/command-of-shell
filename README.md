# javac命令行  

javac用于编译java源文件，生成.class文件。形式如下`javac [option] source`  



# 2.java命令行  

java用于运行.class文件。形式如下：`java [option] classname [arguments]`  

其中，Java后面的最终文件不能含有.class，否则会报错
